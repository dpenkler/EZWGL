4.20 WorkAreas

A workArea is a free format lister for display items. It is like a canvas but with limited display capabilities. Each workArea widget has a private geometry manager, which arranges the locations of display items.

Widget Info

Symbolic widget type EZ_WIDGET_WORK_AREA
Default class name "WorkArea"
Default instance name "workArea"

Symbolic widget type EZ_WIDGET_IWORK_AREA
Default class name "IWorkArea"
Default instance name "iWorkArea"

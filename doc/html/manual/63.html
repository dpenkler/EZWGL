5.8 Disable/Enable a Widget

EZwgl provides two pairs of routines to control the state of EZ widgets. A widget can be disabled or de-activated. A disabled widget will not handle user inputs. A de-activated widget will no longer be displayed until re-activated and re-displayed. 

void EZ_DisableWidget(EZ_Widget *widget);
void EZ_EnableWidget(EZ_Widget *widget);

void EZ_DisableWidgetTree(EZ_Widget *widget);
void EZ_EnableWidgetTree(EZ_Widget *widget);



void EZ_DeActivateWidget(EZ_Widget *widget);
void EZ_ActivateWidget(EZ_Widget *widget);

void EZ_DeActivateChildren(EZ_Widget *widget);
void EZ_ActivateChildren(EZ_Widget *widget);

void EZ_DeActivateSibling(EZ_Widget *widget);
void EZ_ActivateSibling(EZ_Widget *widget);

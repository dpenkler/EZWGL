11 Trees and Direcory Trees

A tree is a special arrangement of display items. A directory tree is a graphical representation of the unix file system. A tree may be displayed in three styles: tree, list tree or branched list tree. Tree displaying is controled by three parameters, indent, line spacng and draw-branch. These attributes is set by the command EZ_TreeSetCharacter.

4.36 Histogram

A histogram displays an array of vertical or horizontal bars, representing an array of floating point values. The size of the array, the number of bins is controled by the EZ_HISTOGRAM_SAMPLE resource. If this resource value is 0, it displays a dynamic array. The range of the histogram is controlled by the EZ_BAR_RANGE resource. The default range is [0, 100]. The width of the bars is controlled by the EZ_BAR_WIDTH resource. The colors of the bars is set by the EZ_BAR_COLOR_N resource.

Widget Info

Symbolic widget type EZ_WIDGET_HISTOGRAM
Default class name "Histogram"
Default instance name "histogram"

Widget Interface Routines

void EZ_SetHistogramValues(EZ_Widget *widget, float *values, int nvalues);

set histogram bin values.

void EZ_SetHistogramValue(EZ_Widget *widget, float value);

set the value for the next bin. Used in progressing histograms.

float EZ_GetHistogramValue(EZ_Widget *widget, int n);

retrieve the nth value.

int EZ_GetHistogramSelection(EZ_Widget *widget, float *value_return);

retrieve the last selected bin and its value.

float EZ_SetHistogramRange(EZ_Widget *widget, float frm, float to);

set the range.

void EZ_SetHistogramColors(EZ_Widget *widget, unsigned long *colors, int ncolors);

set bar colors, ncolors is should be smaller than 12.

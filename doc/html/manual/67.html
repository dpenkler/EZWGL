5.12 Grab a Widget

For dialogue boxes, one usually wants to suspend the application until the user responds to the dialogue. In particular, one needs to suspend event processing on all widgets except the dialogue. The latter can be achieved by using EZ_SetGrab. 


void  EZ_SetGrab(EZ_Widget *widget);
void  EZ_ReleaseGrab(void);

EZ_SetGrab sets a grab on widget. It informs the EZ event dispatcher to drop all events not associated with widget or its descendants. The grab, if set succesfully, will remain in effect until either

  • the widget becomes unviewable. I.e., the widget window or one of its ancestor's window is unmapped; or

  • a call to EZ_ReleaseGrab.

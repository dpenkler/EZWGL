4.23 Status Meters

A status meter is like a progress bar. It has either a vertical or a horizontal orientation. A status meter is capable of displaying five distinct quantities simultaneously.

Widget Info

Symbolic widget type EZ_WIDGET_STATUS_METER
Default class name "StatusMeter"
Default instance name "statusMeter"

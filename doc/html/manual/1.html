1.1 Conventions

  • Naming All functions and constants in EZwgl begin with the prefix EZ_ to avoid name clashing. In this document, EZwgl functions are printed in a typewriter font.

  • Screen Distance Screen distance in the EZ widget library is measured in pixels.

  • Coordinates Coordinates are always parent relative, measured from the Upper-Left corner of the widget window.

  • Colors Colors are specified by either the symbolic names, like "red", or hexdecimal strings like "#7f00ae".

  • Fonts The EZ widget library adopts X's convention in naming fonts.

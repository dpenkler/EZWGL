EZ_GetWidgetTileOrigin

 
NAME

       EZ_GetWidgetTileOrigin  -  return  the  tile origin of the
       background GC.



SYNOPSIS

       #include <EZ.h>

       void EZ_GetWidgetTileOrigin(EZ_Widget *widget, int *x, int  y)




ARGUMENTS

       widget  Specifies an EZ widget.

       x, y  Specifies the returns of the tile origin.




DESCRIPTION

       EZ_WidgetTileOrigin returns origin of the background tile.

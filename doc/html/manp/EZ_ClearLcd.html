EZ_ClearLcd

 
NAME

       EZ_ClearLcd - clear a LCD widget



SYNOPSIS

       #include <EZ.h>

       void EZ_ClearLcd(EZ_Widget *lcd)



ARGUMENTS

       lcd  Specifies a LCD widget.



DESCRIPTION

       EZ_ClearLcd erases the display of an LCD widget.

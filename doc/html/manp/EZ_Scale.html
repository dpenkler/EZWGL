EZ_Scale

 
NAME

       EZ_Scale - multiply the current matrix by a scaling matrix



SYNOPSIS

       #include <EZ.h>

       void EZ_Scale(float x, float y , float z)




ARGUMENTS

       x,y,z Specify three scaling factors.



DESCRIPTION

       EZ_Scale multiplies the current matrix by the matrix

               diag[x,y,z,1]

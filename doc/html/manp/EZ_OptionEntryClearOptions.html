EZ_OptionEntryClearOptions

 
NAME

       EZ_OptionalEntryClearOptions,    EZ_OptionalEntryAppendOp-
       tion,  EZ_OptionalEntryDeleteOption,  -   manipulate   the
       option list for OptionalEntry widget



SYNOPSIS

       #include <EZ.h>

       void  EZ_OptionalEntryClearOptions( EZ_Widget *widget)
       void  EZ_OptionalEntryAppendOption( EZ_Widget *widget, char *option)
       void  EZ_OptionalEntryDeleteOption( EZ_Widget *widget, char *option)


ARGUMENTS

       widget  Specifies an OptionalEntry widget

       option  Specifies a string



DESCRIPTION

       These  functions manipulate the option list of an Optiona-
       lEntry widget.  EZ_OptionalEntryClearOptions  removes  all
       options from the list, EZ_OptionalEntryAppendOption append
       'option' at the end of the option list and  EZ_OptionalEn-
       tryDeleteOption  deletes  the  specified  option  from the
       option list.

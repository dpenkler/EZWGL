EZ_CMove2f

 
NAME

       EZ_CMove2f,  EZ_CMoveScreen2f - update the current charac-
       ter position



SYNOPSIS

       #include <EZ.h>

       void EZ_CMove2f(float x, float y)
       void EZ_CMoveScreen2f(float x, float y)



ARGUMENTS

       x, y Specifies the coordinate of a point.



DESCRIPTION

       EZ_CMove2f(x,y)  is  equivalent  to   EZ_CMove3f(x,y,0.0).
       EZ_CMoveScreen2f(x,y)    is    equivalent   to   EZ_CMove-
       Screen3f(x,y,0.0).

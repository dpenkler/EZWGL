EZ_TreeFindNode

 
NAME

       EZ_TreeFindNode  - search a node in a tree



SYNOPSIS

       #include <EZ.h>

       EZ_TreeNode  *EZ_TreeFindNode(EZ_TreeNode *root, EZ_TreeNode *node)




ARGUMENTS

       root  Specifies the root node of a tree.

       node  Specifies a node.



DESCRIPTION

       EZ_TreeFindNode  searches the tree rooted at root for node
       node and returns the result.

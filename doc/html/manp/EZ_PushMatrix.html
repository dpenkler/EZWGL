EZ_PushMatrix

 
NAME

        EZ_PushMatrix - push down the current matrix stack



SYNOPSIS

       #include <EZ.h>

       void EZ_PushMatrix(void)




DESCRIPTION

       EZ_PushMatrix pushes down the current matrix stack, dupli-
       cating the current matrix on the stack.

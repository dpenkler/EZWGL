EZ_GlobHiddenFiles

 
NAME

       EZ_GlobHiddenFiles  - set an internal flag for globbing




SYNOPSIS

       #include <EZ.h>

       int EZ_GlobHiddenFiles(int yesNo)



ARGUMENTS

       yesNo Specifies  a boolean value.



DESCRIPTION

       EZ_GlobHiddenFiles sets the showHidden flag for the inter-
       nal glob procedure. It return the value of this flag.

EZ_WidgetIsViewable

 
NAME

       EZ_WidgetIsViewable   -  query  if  a  widget is currently
       viewable



SYNOPSIS

       #include <EZ.h>

       int  EZ_WidgetIsViewable(EZ_Widget *widget)




ARGUMENTS

       widget  Specifies a widget.




DESCRIPTION

       EZ_WidgetIsViewable returns 1 if widget is currently view-
       able and returns 0 if it is not.

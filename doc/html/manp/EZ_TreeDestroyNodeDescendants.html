EZ_TreeDestroyNodeDescendants

 
NAME

       EZ_TreeDestroyNodeDescendants - destroy all descendants of
       a tree node



SYNOPSIS

       #include <EZ.h>

       void EZ_TreeDestroyNodeDescendants(EZ_TreeNode *node)



ARGUMENTS

       node  Specifies a tree node.




DESCRIPTION

       EZ_TreeDestroyNodeDescendants destroies all descendants of
       a tree node.

EZ_MatrixMode

 
NAME

       EZ_MatrixMode - specify the current matrix stack



SYNOPSIS

       #include <EZ.h>

       void EZ_MatrixMode( int mode)



ARGUMENTS

       mode  specifies  a  matrix stack. Must be EZ_PROJECTION or
       EZ_MODELVIEW.



DESCRIPTION

       EZ_MatrixMode specifies which matrix stack is  the  target
       for subsequent matrix operations.

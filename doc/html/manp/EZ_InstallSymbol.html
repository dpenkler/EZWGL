EZ_InstallSymbol

 
NAME

       EZ_InstallSymbol,  EZ_RetrieveSymbol  - install/retrieve a
       symbolic constant to/from the resource database



SYNOPSIS

       #include <EZ.h>

       int EZ_InstallSymbol(char *name, EZ_Value value)

       EZ_Value *EZ_RetrieveSymbol(char *name)



ARGUMENTS

       name  Specifies a name.

       value  Specifies a value to be associated with name.




DESCRIPTION

       EZ_InstallSymbol installs the key-value pair (name, value)
       into  an internal lookup table.  It is a tool for convert-
       ing resource values to other  data  types.  This  function
       returns -1 if it fails.

       value is defined by the following data structure.
       typedef struct {
          int        type; /* EZ_INT, EZ_FLOAT, EZ_POINTER or EZ_ULONG */
          union {
            int   i;
            float f;
            void  *p;
            ulong l;
          }          value;
       } EZ_Value;

       EZ_RetrieveSymbol  retrieves  and  returns  the  value for
       name.  If name is not found in the table, it returns NULL.
